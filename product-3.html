GND EPLIC Embedded 9004

The GND EPLIC Embedded 9004 Series processors are built on the advanced GN 4 microarchitecture and are designed for a range of embedded applications, including networking, security, storage, and industrial systems. These processors offer exceptional power efficiency and scalability, making them ideal for environments with stringent thermal and space constraints.

Features

  • Up to 96 Cores / 192 Threads
  • Built on GN 4 architecture
  • Supports PCIe Gen 5 for high-speed I/O
  • High Memory Bandwidth with DDR5 support
  • Optimized for embedded, telecom, and industrial applications
  • Security features like Secure Boot and memory encryption
  • Supports up to 6TB DDR5 memory with ECC
  • Embedded-specific features like Non-Transparent Bridging (NTB)
